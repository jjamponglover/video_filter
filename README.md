# ◾ Introduce
* FPGA(Cora z7) 보드를 이용 DC모터를 사용하여 선풍기를 구현
* Verilog언어를 Vivado 프로그램을 사용하여 설계

# ◾ Basic Features
* 전원 및 바람 세기 조절 3단계
* LED 밝기 3단계
* 타이머 1 3 5 시간 조절
* 바람세기 및 LED밝기 버튼 오래 누르면 OFF
